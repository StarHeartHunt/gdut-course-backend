# GDUT Course

又一个课表小程序，但是开源。

本仓库为小程序 GDUT Course 的 **后端** 项目仓库。

## 安装项目

```bash
$ pnpm install
```

## 运行服务

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## 测试

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
